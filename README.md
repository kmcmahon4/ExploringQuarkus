## Running the application in dev mode with hot reload

 ```shell script 
 make build_dev
 ```

## Building the docker container

 ```shell script 
 make build
 ```

## Running the docker container

 ```shell script 
 make run
 ```

## DEPENDENCIES:
### - Java 21
### - Maven 3.9.5
### - IDE that Supports Java 21 (e.g. Intellij 2023.3 EAP)
### - Docker 4.25.0
### - MySQL 8.0
